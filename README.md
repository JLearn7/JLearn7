## Gambling Prevention App

This repository includes a simple web-based tool called **SafeStreak** designed to help users avoid gambling urges. It features:

- A calming breathing exercise
- A lightweight chat coach that offers motivational messages
- A journaling area that stores entries in your browser

Open `gambling-prevention-app/index.html` in your browser to try it out.
